📋 CSV Format Requirements

  • Required columns: student_id, first_name, last_name
  • Optional columns: rfid_number, course, balance
  • RFID number is used for tap ID purposes (e.g., card scanning)
  • Balance should be in decimal format (e.g., 150.50)
  • Course field is optional
  • Balance will be displayed in Philippine Peso (₱) with comma formatting
  • Make sure your CSV file uses UTF-8 encoding (or Latin-1/Windows-1252 for older files)
  • If you get encoding errors, try saving your CSV file as UTF-8 in Excel or your text editor

📄 Sample CSV Format

student_id,first_name,last_name,rfid_number,course,balance STU001,John,Doe,RFID123456,Computer Science,1500.50 STU002,Jane,Smith,RFID123457,Business Administration,2500.00 STU003,Mike,Johnson,RFID123458,Engineering,750.25 STU004,Sarah,Wilson,RFID123459,Medicine,3200.75
